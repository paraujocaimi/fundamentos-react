# Fundamentos React

Projeto criado aprender react. 


</br>

## Instalar dependencias

Inserir o seguinte comando no cmd 
```
    yarn
```
</br>

## Para executar o projeto

Inserir o seguinte comando no cmd

```
    yarn start
```

</br>
